# vapor

A new Flutter project.

## AI text improvement

AI text improvement reads OpenAI settings from `.env`:

```env
OPENAI_API_KEY=sk-...
OPENAI_MODEL=gpt-5.4-nano
```

`.env` is ignored by git and bundled as a local Flutter asset. `.env.example`
shows the expected keys.
